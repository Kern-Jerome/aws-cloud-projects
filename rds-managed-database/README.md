# Amazon RDS Managed Database Deployment

## Objective
Deploy a managed relational database using Amazon RDS to understand database services in the cloud.

## Services Used
- Amazon RDS
- AWS EC2 (conceptual comparison)

## What I Did
- Launched a Free Tier Amazon RDS instance
- Selected a relational database engine
- Reviewed backup, availability, and maintenance settings
- Compared managed vs self-managed database responsibilities
- Terminated the instance after validation

## Key Concepts Learned
- Managed services vs infrastructure management
- High availability and automated backups
- Cloud database scalability and reliability

## Outcome
Gained hands-on experience with AWS-managed database services and operational best practices.
